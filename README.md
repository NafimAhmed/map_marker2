# dynamic_marker

A Flutter package for creating dynamic/custom map markers.
You can easily generate marker widgets with text, images, and badges,
and use them as bitmap descriptors on Google Maps or other map widgets.

## Features

- Generate marker widgets dynamically
- Support for adding text, icons, and images
- Convert widgets to BitmapDescriptor for Google Maps

## Installation

Add this to your `pubspec.yaml`:

```yaml
dependencies:
  dynamic_marker: ^0.0.1
